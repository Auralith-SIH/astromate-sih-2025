1. Research Foundation & Problem Validation
Our solution is grounded in challenges identified by NASA and ISRO.

Cognitive Fatigue & Human Error
Source: NASA Human Research Program

Relevance: Validates performance risks from fatigue and isolation, directly addressed by CogTwin.

Link: https://humanresearchroadmap.nasa.gov/

Astronaut Physiological Standards
Source: ISRO Human Space Flight Centre (HSFC)

Relevance: Informs physiological baselines (sleep, heart rate) for CogTwin’s fatigue model.

Link: https://www.isro.gov.in/HumanSpaceFlight.html

Radiation Risk Modeling
Source: NASA OMNIWeb Plus Database

Relevance: Historical solar particle data for ShieldMate’s radiation risk algorithm.

Link: https://omniweb.gsfc.nasa.gov/

Real-Time Solar Monitoring
Source: NASA Solar Dynamics Observatory (SDO)

Relevance: Source for simulated real-time solar flare data in our prototype.

Link: https://sdo.gsfc.nasa.gov/data/

2. Technical Implementation
AI/ML Core
TensorFlow Lite: For lightweight, on-device fatigue prediction.

Scikit-learn: For data analysis and preprocessing.

Purpose: Enables real-time inference without cloud dependency.

Task Automation
Google OR-Tools: For efficient automatic rescheduling of astronaut tasks.

Link: https://developers.google.com/optimization

Workload Assessment
NASA Task Load Index (NASA-TLX): Framework for mental workload assessment.

Citation: Hart, S. G., & Staveland, L. E. (1988). Advances in Psychology, 52, 139-183.

Development Stack
Frontend: React.js, Tailwind CSS, Chart.js, Web Speech API (for voice alerts)

Backend: Python, Flask, Pandas, NumPy

Deployment: Simulated data, Ngrok, GitHub
